## Project Overview
This SQL project implements a cab booking management system using MySQL.
The database structure, tables, and sample data are defined in the main SQL file,
while the problem statements are provided separately.

## Files Description
- PROJECTDATABASE.sql  
  Contains database creation, table definitions, and data insertion scripts.
  This is the main project implementation file.

- PROJECTSQL.sql  
  Contains business problem statements and SQL questions solved using the database.

## Objectives
- Design a relational database using MySQL
- Apply SQL joins, subqueries, and aggregations
- Solve real-world business problems using SQL

## Database Components
- Customers
- Drivers
- Bookings
- Payments

## SQL Concepts Used
- INNER JOIN, LEFT JOIN
- GROUP BY, HAVING
- Subqueries
- Aggregate functions
- Date functions

## How to Run the Project
1. Open MySQL Workbench
2. Execute `PROJECTDATABASE.sql` to create database and tables
3. Refer to `PROJECTSQL.sql` for problem statements and run queries
